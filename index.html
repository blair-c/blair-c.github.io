rivals.academy

Official website + API for the Rivals Academy gaming community which links novices with top players for coaching. Built with Hugo.

• Library

Visualizations + detailed stats for all characters, attacks, and stages in the game. Data exposed as open JSON API, generated alongside page HTML from edited YAML.

• Lectures

General + character strategy guide articles in English and Spanish. Writing of new guides facilitated through Discord with 20+ contributors so far.

Dev Notes

  • 100/100/100/100 Google Lighthouse Score (on most pages)
  • Automatic site builds and deployments with GitHub Actions CI/CD
  • Automated processing of moveset visualizations with ImageMagick

Mentorbot 3.0

Discord app for Rivals Academy gaming community that serves over 50,000 users across 300+ community hubs. Built with discord.py.

• /invite

Created to drastically speed up the Steam matchmaking process, allowing users to seamlessly invite other members without leaving Discord. Uses Redis Database to persistently store links between users’ Discord and Steam accounts.

• Game Info Commands

Mentorbot's original and most popular commands serve as a quick reference for hitboxes or other game info with a link to the full data on rivals.academy.

• Mentor Commands

Gives users a quick view into available volunteer coaches for their character, one of the core draws of the Academy. Admin commands give these volunteers the ability to manage how and when they appear on these lists.

SteamJoin

Utility site inspired by Mentorbot's /invite command that also functions as a standalone service. Built with Go, go-chi, and TailwindCSS.

Parses Steam API responses and uses Steam browser protocol to redirect into Steam on success.

Dev Notes

  • Utilized go-chi dynamic routing and JavaScript to unify search and redirect implementation
  • Enabled logging + rate limiting with standard library compatible go-chi middleware
  • Styled single Go Template file with Tailwind CSS
